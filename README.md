Project Overview

The study area is located off the northwestern coast of Africa, in the eastern Atlantic Ocean, near the coastlines of Morocco and Western Sahara.

The project combines both terrestrial and marine data, allowing for:
- terrain morphology analysis
- seabed bathymetry analysis

Key Characteristics:
- Elevation range: +122 m to -2750 m
- Presence of steep bathymetric gradients
- High morphological diversity


Tools & Technologies

- QGIS – spatial data processing and visualization
- ParaView – advanced 3D data analysis and visualization
- OpenStreetMap – base spatial data source
- TIN (Triangulated Irregular Network) – interpolation method


Workflow

The project consists of several key stages:

1. Study Area Visualization
- Mapping the selected region
- Integration of land and marine datasets

2. Interpolation (TIN)
- Generation of a continuous surface from discrete elevation data
- Creation of terrain/bathymetry model in QGIS

3. Azimuth Analysis
- Calculation of surface orientation (aspect)
- Visualization in:
  - QGIS
  - ParaView

4. Surface Orientation Clustering
- Grouping of surface normals (XYZ-N)
- Identification of dominant orientations

5. Stereonet Analysis
- Visualization of:
  - surface orientation
  - slope directions
- Interpretation of spatial patterns

6. Slope Direction Analysis
- Clustering of slope directions (XYZ-D)
- Identification of dominant slope trends

7. Geomorphon Classification
- Automatic landform classification
- Identification of terrain features (e.g., ridges, valleys, slopes)


Results

The project produced:
- Interpolated terrain/bathymetry models
- Azimuth (aspect) maps
- Surface orientation clusters
- Stereonet diagrams
- Slope direction analysis
- Geomorphon-based landform maps

These outputs allow for a deeper understanding of:
- terrain structure
- seabed morphology
- directional patterns in surface geometry
